## Rodando com Docker

Pré-requisito:
- Docker e Docker Compose instalados.

Comandos:
1. Construir a imagem:
   ```bash
   docker compose build
   ```
2. Subir a aplicação:
   ```bash
   docker compose up
   ```
3. Acessar no navegador:
   ```text
   http://localhost:8000
   ```

Para parar:
```bash
docker compose down
```
